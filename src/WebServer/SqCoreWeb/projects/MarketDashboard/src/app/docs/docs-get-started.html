User Guide: There are 3 tools (services) in the MarketDashboard. Click the Tools menu with the Red triangle on the top.

  1. MarketHealth: showing a realtime info about tickers, such as QQQ (Nasdaq100), SPY (SP500), GLD (Gold), TLT (20y Treasuries), VXX (VIX volatility), UNG (UnitedStates Natural Gas), USO (United States Oil).
    1. If you click the ticker, it will go to the TradingView chart.
    2. Play with the combo-boxes for more info. With this tool, you can see real-time for example that
      1. YearToDate (YTD), in the SPY (S&P 500 index), there was a -34% maximum drawdown at the deepest panic. However, now we are only in a -16% current drawdown (from highest high).
      2. SPY: currently we are+25% away from the panic low (the terminology is "we have a +25% DrawUp ")
      3. YearToDate (YTD) SPY return is -13%, but Nasdaq (QQQ) return is actually positive: +0.75%. Wow! Surprising! :) In this pandemic times.
    3. There is a semi-transparent tooltip for both rows
    4. Every number is colour coded, red for negative, green for positive.
    5. Prices are realtime not only during regular trading hours (update frequency: 3sec), but also pre-market and post-market (update frequency: 60sec (to avoid YahooFinance IP ban)).
  2. Tools menu: QuickfolioNews
    1. There are general news and company-specific news.
    2. There are preview panels for both. Preview of the news changes on mouse hover. You don't need to click.
    3. Updated real-time every minute.
  3. Tools menu: CatalystSniffer
    1. It shows Brooke's ribbon design as a static image at the moment.
