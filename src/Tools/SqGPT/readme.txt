
1. Which LLM model to use.

Change the 'string modelPath = "g:\\LLM\\llama-2-7b.ggmlv3.q4_0.bin";' in Program.cs to use the your local LLM model file.

We suggest using the LLama2 (uncensored) Base version model (llama-2-7b.ggmlv3.q4_0.bin), because the chat version is censored.
It can be downloaded at https://huggingface.co/TheBloke/Llama-2-7B-GGML
(Click 'File and Versions' tab and download llama-2-7b.ggmlv3.q4_0.bin (3.79GB))
